public：静态文件。用来存放项目静态文件目录如js,css以及图片。
routes：路由文件。路由主要定义 url 和 资源 的映射关系 ( 一一对应关系 ),
主要用来接收前端发送的请求 响应数据给前端
views：后端模版文件。
app.js：入口文件。
package.json：工程信息和模块依赖。
